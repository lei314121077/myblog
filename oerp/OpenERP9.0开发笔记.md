## OERP零散
  
* 替换logo为自己公司的logo
    
    ```
      # 进到目录
        
        $ cd ./openerp/addons/base/static/img
      
      # 替换名称为 logo_white.png 的logo图片
        
        $ sudo vm ./openerp/addons/base/static/img/logo_white.png   logo_white.png   
    
    ```
    
    > 更换完，如果加载之后还是原来的logo那么，请关闭你的浏览器然后再清一下缓存。
