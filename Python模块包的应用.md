# Virtualenv（虚拟环境）
  
  > virtualenv通过创建独立Python开发环境, 来解决项目之间环境的依赖以及版本之间内存在的间接权限类的问题。
  
  * 安装
  
    ```shell
      $ pip install virtualenv
      $ sudo pip install virtualenv
      $ easy_install virtualenv
    
    ```
  
  * 正确的姿势 (基本使用)
  
    ```shell
      odoo git:(9.0)x     
    ```
  
  * 觉醒（激活）
  
  * 闭关 （关闭）
  
  * 唐伯虎点秋香（指定目标版本）
  
  * 明媒正娶 （生成可打包的环境）
  
  * 求援 （帮助）
  
  [Virtualenv官网文档](http://virtualenv.readthedocs.org/)
