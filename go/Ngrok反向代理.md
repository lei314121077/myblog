# [Ngrok官网](https://ngrok.com/)，[github](https://github.com/inconshreveable/ngrok)


## 用法 

   * 安装
   
      ```go
        # mac install 
        $ brew install ngrok
        # 认证
        $ ngrok authtoken <your-token>
        # run
        $ ngrok http 8080
        # 然后等链接到服务器会获得ngrok的地址
      ```
