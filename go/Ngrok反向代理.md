# [Ngrok官网](https://ngrok.com/)，[github](https://github.com/inconshreveable/ngrok)

   Ngrok服务的核心功能是能够将你本机的HTTP服务（站点）或TCP服务，通过部署有ngrok服务的外网伺服器暴露给外网访问！

## 用法 

   * 安装
   
   ```go
        # mac install 
        $ brew install ngrok
        # 认证
        $ ngrok authtoken <your-token>
        # run
        $ ngrok http 8080
        # 然后等链接到服务器会获得ngrok的地址
   ```
